# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x.x   | ✅ Yes    |

## Reporting a Vulnerability

**Please do NOT report security vulnerabilities through public GitHub issues.**

Instead, email: `security@YOUR_DOMAIN.com`

Include:
- Type of vulnerability
- File paths involved
- Steps to reproduce
- Potential impact

You will receive a response within 48 hours. If confirmed, a patch will be released
within 7 days and you will be credited in the release notes.

## Privacy

CodePulse stores **zero data externally**. All analysis results and activity logs
are stored in VS Code's local `globalState` on your machine only.
No telemetry. No analytics. No network requests.
